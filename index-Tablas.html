Tablas.

Ideas principales:

Las etiquetas:

  1. <table>.
  2. <thead>.
  3. <tr>.
  4. <th>.
  5. <tbody>.
  6. <td>.
  7. <caption>.
  8. <tfoot>.

Atributos:

  • rowspan.
  • colspan.
  • border.

Apuntes de clase:

Table: Permite organizar información en filas y columnas dentro de celdas. Las etiquetas para hacer una tabla son:

  • <table>: Define la tabla (etiqueta padre).
  • <thead>: Define el encabezado de la tabla.
  • <tr>: Define una fila.
  • <th>: Define una celda de encabezado.
  • <tbody>: Contiene el cuerpo de la tabla.
  • <td>: Define una celda con datos.
  • <caption>: Etiqueta opcional para poner un contexto (opcional).
  • <tfoot>: El pie de la tabla (opcional).

Atributos:

  • rowspan: Combina dos o más celdas de manera vertical, que se utiliza en las etiquetas <th> y <tr>.
  • colspan: Combina dos o más celdas de manera horizontal, que se utiliza en las etiquetas <th> y <tr>.
  • border: Define el grosor de un borde para la tabla.

Resumen de la clase:

Las tablas en HTML se construyen con las etiquetas <table>, <thead>, <tr>, <th>, <tbody>, <td>, <caption> y <tfoot>, que definen la estructura y contenido de la tabla. Los atributos rowspan y colspan permiten combinar celdas vertical y horizontalmente, mientras que border controla el grosor del borde de la tabla.
