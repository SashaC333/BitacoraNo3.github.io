Formularios.

Ideas principales:

Etiquetas principales:

  • <form>
  • <input>
  • <label>
  • <textarea>
  • <button>
  • <select>

Apuntes de clase:

  • <form>: Contenedor principal del formulario.
  • <input>: Campo de entrada de datos, y usa type.
  • <label>: Proporciona una descripción para los campos, mejorando la accesibilidad.
  • <textarea>: Área de texto para introducir varias líneas de contenido.
  • <button>: Botón personalizable para realizar acciones en el formulario.
  • <select>: Menú desplegable para seleccionar opciones.

Tipos de <input>

  • text: Entrada de texto .
  • password: Entrada para contraseñas.
  • email: Entrada para direcciones de correo electrónico.
  • number: Entrada para números.
  • date: Entrada para seleccionar una fecha.
  • time: Entrada para seleccionar una hora.
  • file: Subida de archivos.
  • checkbox: Opción de selección múltiple.
  • radio: Selección única entre varias opciones.
  • submit: Botón para enviar el formulario.
  • reset: Botón para reiniciar los campos.
  • range: Selector deslizante para valores en un rango.
  • color: Selector de color.
  • tel: Entrada para números de teléfono.
  • url: Entrada para URLs.
  • search: Campo de búsqueda.

Atributos:

  • action: URL a la que se envían los datos del formulario.
  • method: Método de envío (GET o POST).
  • name: Nombre único para identificar el campo.
  • id: Identificador único del campo.
  • value: Valor inicial del campo.
  • placeholder: Texto de ayuda dentro del campo.
  • required: Hace obligatorio un campo.
  • readonly: Campo de solo lectura.
  • disabled: Desactiva el campo.
  • maxlength: Número máximo de caracteres permitidos.
  • cols y rows: Limita el número de caracteres por columnas y renglones.

Resumen de la clase:

El <form> es el contenedor principal de un formulario, y dentro de él se usan diversos elementos como <input> (para distintos tipos de entrada), <label> (para describir los campos), <textarea> (para áreas de texto), <button> (para botones), y <select> (para menús desplegables). Los tipos de <input> incluyen opciones como text, password, email, checkbox, radio, submit, entre otros, que deciden la conducta del campo.

Y los atributos importantes de los formularios son action (URL de destino), method (GET o POST), name, id, value, placeholder, required, readonly, disabled, y maxlength, que controlan la interacción y validación de los datos.
